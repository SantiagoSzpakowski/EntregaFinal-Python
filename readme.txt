Descripción del Proyecto:
Página Web destinada a usuarios que deseen publicar sus proyectos.

Si el usuario no inicia sesion solamente podra ver las páginas "Inicio", "Acerca de mi", "Proyectos" pero no podra agregar nuevos proyectos.

Si el usuario inicia sesión, una vez la página valide la autenticación del usuario, este será redirigido al inicio de la página web y en la página "Proyectos" podra publicar sus proyectos. También puede editar sus datos de usuario como nombre, mail, contraseña y avatar.

El usuario administrador puede realizar las siguientes accciones adicionales:
- Editar cualquier proyecto sin importar quien lo subió
- Agergar o borrar la imagen de cualquier proyecto
- Borrar proyectos
